## Features
- Inventory tracking for food and supply items
- User authentication and role-based access
- Low stock alerts
- Expiration date tracking
- Report generation

## Technologies
- Java 21
- Maven
- JUnit 5
- Apache Derby (Project 2)

## How to Run
1. Clone the repository
2. Open in NetBeans 23
3. Run the main class: `com.cafe.inventory.CafeSystem`
